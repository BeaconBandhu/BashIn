The Mechanism

Watch it learn you.

01 / OBSERVE

It watches, silently.

The cursor overlay runs above every application. It sees what app you're in, what you're reading, where you pause, what you re-read. Passive. Zero interruptions. Sub-50ms latency.

const learn = async () => {
  await observe(screen);
  const pattern = map(behavior);
  return train(pattern);
}
02 / MAP

Your knowledge, mapped.

Every interaction becomes a node. Your Obsidian vault grows in the background. Graphify draws the connections — topics, patterns, relationships — as you work. Automatically.

Graphify Obsidian Vault Auto-sync
03 / VERIFY

It asks before it assumes.

Once a week, a quiet digest: "Here's what I learned about how you work. Confirm or correct." Your model is built on verified truth — not guesses. You stay in control.
